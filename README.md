# 💳 AML Credit Card Fraud Detection & Monitoring Dashboard

## 📌 Project Overview

This project focuses on analyzing credit card transaction data to identify fraudulent and high-risk transactions using **Excel, SQL, and Power BI**. The objective is to simulate a real-world **Anti-Money Laundering (AML) and Fraud Monitoring System** used by banks and financial institutions.

The dashboard provides actionable insights into fraud patterns, risk levels, and transaction behavior, enabling compliance teams to detect and investigate suspicious activity efficiently.

---

## 🎯 Project Objectives

* Detect fraudulent transactions using risk indicators
* Apply a risk-based approach to classify high-risk transactions
* Monitor fraud trends and fraud percentage
* Build an interactive dashboard for AML monitoring
* Simulate real-world AML investigation workflow

## 📊 Dataset Information

**Source:** Kaggle – Credit Card Fraud Detection Dataset

**Dataset Features:**

* 284,807 total transactions
* 492 fraudulent transactions
* Transaction amount
* Fraud label (Class: 0 = Genuine, 1 = Fraud)
* Anonymized transaction features
